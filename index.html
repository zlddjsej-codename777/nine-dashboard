DocumentBuilderFactory dbFactory = DocumentBuilderFactory.newInstance(); DocumentBuilder dBuilder = dbFactory.newDocumentBuilder(); Document dDocument = dBuilder.parse(urlBuilder.toString()); dDocument.getDocumentElement().normalize(); // 측정일시 NodeList nList = dDocument.getElementsByTagName("MSRDT"); Node nNode = nList.item(0); Element eElement = (Element) nNode; System.out.println("파라미터 확인: " + eElement.getTextContent()); // 권역명 NodeList nList2 = dDocument.getElementsByTagName("MSRRGN_NM"); Node nNode2 = nList2.item(0); Element eElement2 = (Element) nNode2; // 측정소명 NodeList nList3 = dDocument.getElementsByTagName("MSRSTE_NM"); Node nNode3 = nList3.item(0); Element eElement3 = (Element) nNode3; // 통합대기환경등급 NodeList nList4 = dDocument.getElementsByTagName("IDEX_NM"); Node nNode4 = nList4.item(0); Element eElement4 = (Element) nNode4; System.out.println(eElement.getTextContent() + " 기준"); System.out.println("(" + eElement2.getTextContent() + ")" + eElement3.getTextContent() + " 날씨는 " + eElement4.getTextContent() + "입니다."); URL url = new URL(urlBuilder.toString()); HttpURLConnection conn = (HttpURLConnection) url.openConnection(); conn.setRequestMethod("GET"); conn.setRequestProperty("Content-type", "application/xml"); //System.out.println("Response code: " + conn.getResponseCode()); /* 연결 자체에 대한 확인이 필요하므로 추가합니다.*/ BufferedReader rd; // 서비스코드가 정상이면 200~300사이의 숫자가 나옵니다. if(conn.getResponseCode() >= 200 && conn.getResponseCode() <= 300) { rd = new BufferedReader(new InputStreamReader(conn.getInputStream())); } else { rd = new BufferedReader(new InputStreamReader(conn.getErrorStream())); } StringBuilder sb = new StringBuilder(); String line; while ((line = rd.readLine()) != null) { sb.append(line); } rd.close(); conn.disconnect(); //System.out.println(sb.toString()); } }
